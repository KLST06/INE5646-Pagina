Dicionário de Tags HTML

Dicionário de Tags HTML

<!DOCTYPE html>
Declaração que define o tipo de documento e versão HTML.
<html>
Elemento raiz que contém todo o conteúdo da página.
<head>
Contém metadados sobre o documento, como o título e links para arquivos de estilo.
<meta>
Define metadados como o conjunto de caracteres e a configuração de visualização.
<title>
Define o título da página exibido na aba do navegador.
<link>
Define relacionamentos com recursos externos, como folhas de estilo CSS.
<nav>
Define uma seção de navegação contendo links para outras partes da página ou site.
<header>
Define um cabeçalho para a página ou seção, geralmente contendo títulos e navegação.
<main>
Define o conteúdo principal da página, excluindo cabeçalhos, rodapés e barras laterais.
<section>
Define uma seção genérica dentro de um documento, usada para agrupar conteúdo tematicamente.
<ul>
Define uma lista não ordenada.
<li>
Define um item em uma lista.
<a>
Define um link para outra página ou recurso.
<footer>
Define um rodapé para a página ou seção, geralmente contendo informações de contato e direitos autorais.
<address>
Define informações de contato.
